# Images Folder

This folder contains images used in the portfolio website.

## Required Files:

### Profile Image:
- `profile.jpg` or `profile.png` - Your professional profile photo

## How to Add Your Profile Image:

1. Take a high-quality professional photo
2. Save it as `profile.jpg` or `profile.png` in this folder
3. The image will automatically appear in the hero section
4. Recommended size: At least 400x400 pixels
5. Format: JPG or PNG

## Tips:
- Use a clear, professional headshot
- Good lighting and neutral background work best
- Square or circular crop works well
- File size should be under 2MB for web performance

The portfolio will display a placeholder until you add your actual profile image.
